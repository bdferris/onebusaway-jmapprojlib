This a branch / fork of the jmapprojlib at:

http://sourceforge.net/projects/jmapprojlib/

maintained by Bernhard Jenny.  This branch adds the following features:

1) Mavenize the build for easier packing and dependcy management by other
   projects.
2) Add datum conversion support and a "latlong" projection similar to PROJ.4.

The license for jmapprojlib is Apache V2.  Portions of the code have been
ported from PROJ.4, which is released under an MIT license.
